Welcome to my personal **NixOS configuration**. This repository houses a reproducible, declarative system configuration built with **Nix Flakes** and **Home Manager**. It is designed to provide a robust, high-performance environment tailored for development and gaming.

> [!WARNING]
> **Compatibility Notice**: This configuration is highly optimized for my specific hardware (e.g., AMD GPU). Please review the hardware configurations in `hardware/` and adjust them to match your system before applying. Using this configuration as-is on different hardware (especially NVIDIA GPUs) may result in system instability or boot failures.

> [!IMPORTANT]
> **Hardware Configuration**: You **must** use the `hardware-configuration.nix` generated by your own system installation (usually located at `/etc/nixos/hardware-configuration.nix`). Do not use the one provided in this repository, as it contains filesystem UUIDs and kernel modules specific to my machine. Replace it with your own before rebuilding.
